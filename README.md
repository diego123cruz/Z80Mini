# Z80Mini
Z80 Mini

Z80 @4Hhz
RAM 32kb @62256 
ROM 32Kb @28C256

Display 7 Segments 8 digits - OUT_40h_B0-B7 + 74LS145
Sensor IR - IN_40h_B7 (PullUp)
KEY_MONI/BACK - IN_40h_B5 (PullDown)
KEYS_0-F - IN_40h_B3, IN_40h_B4 + 74LS145
